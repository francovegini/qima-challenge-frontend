# QIMA Challenge Frontend

## Dependencies used
```
- React
- Vite
- Antd
```

## How to set up the project

1. Create a .env file and add the env below:
```
VITE_API_URL=http://localhost:8080/api
```

2. Install dependencies (Use your preferred package manager)
```
pnpm install
```

3. Start the project
```
pnpm run dev
```

4. Enjoy the ride :)
